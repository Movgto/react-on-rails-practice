# ğŸ“– Ruby on Rails + React: Hello World <a name="about-project"></a>

**Hello World** is a Ruby on Rails and React app that displays a "Hello World" message on the Home page, this is a practice for learning how to integrate React-Redux features in a RoR application, such as components, slices, routing, global store, and more.

### Key Features <a name="key-features"></a>

- Database management using Rails features: models, controllers, and views.
- React-Redux features: components, routing, global store, and reducers.
- Created a 'Greeting' components which displays a random greeting message from the 'greetings' table created with Rails, this is possible thanks to the action 'index' in the 'GreetingsController' which makes an API endpoint available, responding with the data(array of strings) in json format.
